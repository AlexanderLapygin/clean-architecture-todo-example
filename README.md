# :ballot_box_with_check: Clean Architecture Todo Example
:construction: [WIP] An implementation example of the Todo app using Clean Architecture.

## :arrow_forward: Usage
```bash
# API
$ cd api
$ npm install
$ npm start

# App
$ cd app
$ npm install
$ npm start
```
